📊 Data Analytics Internship – Task 1
Data Immersion & Wrangling

This project is part of my Data Analytics Internship at ApexPlanet Software Pvt. Ltd.

The objective of this task is to explore, clean, and prepare a dataset for further data analysis using Python and Pandas.


📁 Dataset
The dataset used in this project is the Netflix Titles Dataset, which contains information about movies and TV shows available on Netflix.
The dataset includes columns such as:
Show ID

Title

Type (Movie / TV Show)

Director

Cast

Country

Release Year

Rating

Duration



⚙️ Tools & Technologies

Python

Pandas

NumPy

Jupyter Notebook



🔎 Steps Performed

1️⃣ Data Loading
Loaded the dataset using Pandas and examined the first few rows to understand the structure of the data.


2️⃣ Data Exploration

Checked:

Dataset shape

Data types

Missing values

Duplicate records


3️⃣ Data Cleaning

Prepared the dataset by ensuring consistent formatting and making it ready for further analysis.
